# 💰 MERN Interest Tracker

A full-stack web application to calculate interest and manage multiple user accounts efficiently. This app allows users to track financial data, perform interest calculations, and manage transactions in a structured way.

---

## 🚀 Features

* 📊 Calculate Simple Interest
* 👤 Manage Multiple Accounts
* 💵 Add & Track Transactions
* 📁 View Account Details
* 🧾 Maintain Financial Records
* ⚡ Fast and Responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB (via Mongoose)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Praveen-Developer-tech/MERN-interest-tracker.git
cd MERN-interest-tracker
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🌐 Running the App

* Frontend: http://localhost:5173
* Backend: http://localhost:5000

---

## 🔄 How It Works

1. User enters financial details in the frontend
2. React sends API request to backend
3. Backend processes interest calculation
4. Data is stored/retrieved from database
5. Results are displayed in UI
